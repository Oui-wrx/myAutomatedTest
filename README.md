环境准备
- python 官网：https://www.python.org/
  - 安装：百度（配置环境变量）
  - 文档：https://docs.python.org/3.9/tutorial/index.html
- git
  - 安装：百度（配置）
- pycharm
  - 安装：百度
  - 配置git：百度
  - 根目录下：git init
    - 右上角会出现下拉和对勾
    - 点击对勾，显示代码目录，点击右键，可以选择添加到 .gitignore
    - 一般可以忽略.idea 和 venv虚拟环境
